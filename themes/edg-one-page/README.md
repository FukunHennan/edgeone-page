# EdgeOne Page

EdgeOne Page 是一个基于 Hexo 的简洁主题，重新定义了简约、速度和纯净，同时不牺牲功能或设计。其时尚、现代的美学包装了许多有用的功能，将风格和实用性无缝融合。

基于 hexo-theme-keep 的坚实基础，"EdgeOne Page" 提升了风格并整合了有价值的写作工具和插件。它提供了广泛的自定义选项，允许您根据自己的喜好调整每个细节。使用 EdgeOne Page，您的博客体验变得独特而轻松，展示您的个人风格和需求。

## ⛰️ 特性

- 笔记模块
- 友情链接页面
- Mathjax 支持
- 代码块语言显示
- 浅色/深色模式切换
- Font Awesome 6.2.1 Pro（包含 Duotone/Regular/Thin/Sharp 等不同样式）
- 下拉菜单
- 可自定义页脚
- 站点运行时间显示
- 文章标题图片
- Mermaid JS 支持
- SEO 友好
- Aplayer 支持
- 说说功能支持
- 可自定义字体
- 包含 Tailwind CSS

## ☁️ 安装

安装 EdgeOne Page 主题的最简单方法是使用 **npm**（在您的 hexo 版本达到 `5.0`+ 之后）

```sh
$ cd your-hexo-site
$ npm install edg-one-page@latest
```

另一种方法是通过 **git clone**

```sh
$ cd your-hexo-site
$ git clone https://gitee.com/chenfukun/edg-one-page.git themes/edg-one-page
```

安装后，前往您的 hexo 站点的 `_config.yml` 并设置

```yaml
theme: edg-one-page
```

## ⏫ 更新

要更新 EdgeOne Page，您可以运行相同的 **npm** 命令

```sh
$ npm install edg-one-page@latest
```

如果您通过 **git clone** 安装了主题，则可以通过运行以下命令来更新主题

```sh
$ cd themes/edg-one-page
$ git pull
```
